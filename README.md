# Expense Tracker (Spring Boot)

A simple and efficient Expense Tracker application built using Spring Boot that helps users manage their daily expenses, track spending patterns, and stay financially organized.

# Overview

This project is designed to provide a backend system for tracking income and expenses. It allows users to add, update, delete, and view their financial transactions with ease. The goal of this project is to demonstrate clean backend architecture, REST API design, and database integration using Spring Boot.

# Tech Stack
Backend: Spring Boot
Language: Java
Database: MySQL
ORM: Spring Data JPA (Hibernate)
Build Tool: Maven
API Testing: Postman

# Features
Add new expenses and income entries
View all transactions
Update existing records
Delete transactions
Categorize expenses (e.g., Food, Travel, Bills)
Filter transactions based on date or category
RESTful API design for easy frontend integration
